- Ruta: `./skills/al-alto-nivel/SKILL.md`
- Objetivo: aplicar prácticas AL de alto nivel para Business Central en diseño, validaciones, rendimiento, integración POS/pinpad, eventos (`LSC/OPT`) y pruebas.

  - Se crea un nuevo objeto `.al`.
  - El usuario pida explícitamente esta skill.

## Reglas De Aplicación
- Usar el conjunto mínimo de skills necesarias para cada tarea.
- Si hay varias skills aplicables, empezar por `al-alto-nivel` para fijar criterios de calidad en AL.

- Cuando haya dudas de estándar AL, validar con documentación oficial de Microsoft Learn.
